# Provider Routing Review

Inspect:

- capability detection
- provider selection policy
- fallback behavior
- ordered streaming transport
- cancellation and partial-output handling
- provider-specific secrets or metadata leakage

Output:

- Routing verdict
- Drift evidence
- Fallback concerns
- Streaming or cancellation gaps
