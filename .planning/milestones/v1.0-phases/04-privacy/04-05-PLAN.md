## Trust Boundaries

| Boundary | Description |
|----------|-------------|
| renderer → files IPC | Renderer triggers picker and reads by token; never supplies or receives a path |
| backend → OS dialog | Path comes from the OS picker directly into backend memory |

## STRIDE Threat Register

| Threat ID | Category | Component | Disposition | Mitigation Plan |
|-----------|----------|-----------|-------------|-----------------|
| T-04-13 | Elevation of Privilege | frontend supplying raw path to read | mitigate | files_read_token accepts only a UUID token; no raw_path parameter exists (D-04) |
| T-04-14 | Information Disclosure | raw path returned in metadata | mitigate | FileTokenResponse exposes filename/size/mime only, no directory/full path |
| T-04-15 | Spoofing | non-main window invoking picker | mitigate | policy_check first in both commands |
| T-04-16 | Denial of Service | blocking_pick_file on Tokio thread deadlock | mitigate | files_open_dialog is a sync command on the blocking pool (Pitfall 3) |
</threat_model>
