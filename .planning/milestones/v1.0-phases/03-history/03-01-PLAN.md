## Trust Boundaries

| Boundary          | Description                                                                               |
| ----------------- | ----------------------------------------------------------------------------------------- |
| IPC → storage     | Only typed store methods may call with_conn; IPC handlers must not bypass stores          |
| Schema migration  | Migrations are &'static str literals; savepoint ids are validated by debug_assert         |
| FTS5 search input | FTS5 MATCH query uses parameterized bind (?1) — no SQL interpolation of user query string |

## STRIDE Threat Register

| Threat ID | Category               | Component                                   | Disposition | Mitigation Plan                                                                                                                                                          |
| --------- | ---------------------- | ------------------------------------------- | ----------- | ------------------------------------------------------------------------------------------------------------------------------------------------------------------------ |
| T-03-01   | Tampering              | migrations.rs MIGRATIONS slice ordering     | mitigate    | append-only pattern; debug_assert on id charset; test asserts MIGRATIONS.len() == 3                                                                                      |
| T-03-02   | Information Disclosure | FtsStore::search snippet() output           | accept      | snippet is backend-generated from user's own stored content; no external data; content='messages' external-content table keeps message text in main table, not FTS index |
| T-03-03   | Denial of Service      | WAL checkpoint blocking on busy readers     | accept      | Non-fatal pattern per D-14; checkpoint failure does not block delete response; eprintln warning only                                                                     |
| T-03-04   | Tampering              | FTS5 MATCH query injection                  | mitigate    | All FTS5 queries use SQLite bind parameters (?1) — query string is never interpolated into SQL text                                                                      |
| T-03-05   | Information Disclosure | ConversationRow/MessageRow serde::Serialize | mitigate    | Content crosses IPC only through typed commands (Plan B); these structs are not directly exposed — plan B maps them to typed IPC response types                          |
| T-03-SC   | Tampering              | cargo build deps                            | mitigate    | No new crate deps in this plan (uses existing rusqlite, serde, uuid already in Cargo.toml)                                                                               |
