## Trust Boundaries

| Boundary                          | Description                                                                            |
| --------------------------------- | -------------------------------------------------------------------------------------- |
| backend preview → renderer chrome | Renderer only receives sanitized `srcdoc`; it must not be able to alter sandbox policy |
| chat event → surface navigation   | `ArtifactReady` is advisory UI state, not permission to execute content                |

## STRIDE Threat Register

| Threat ID | Category               | Component                  | Disposition | Mitigation Plan                                                                              |
| --------- | ---------------------- | -------------------------- | ----------- | -------------------------------------------------------------------------------------------- |
| T-05-03   | Tampering              | Artifacts surface controls | mitigate    | keep stop/reload in host chrome outside the iframe so artifact content cannot intercept them |
| T-05-04   | Information Disclosure | frontend preview caching   | mitigate    | store only sanitized preview data and reload from backend on demand                          |
| T-05-SC   | Tampering              | surface navigation path    | mitigate    | navigate through the existing surface store, not through raw route mutation or local storage |

</threat_model>
